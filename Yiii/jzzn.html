彝文檢字表指南

公開 2025年12月27日

  1. 1. 使ひ方
  2. 2. 配列について
  3. 3. 部首について

　彝文檢字表は、彝文字を讀んでゐてわからない字が出て來たときに、讀み（音）を調べるための文字一覽です。Wikipedia Commons の Category:Yi characters by radical に載ってゐるデータを檢索しやすい樣に一頁にまとめたものです。

1. 使ひ方

　檢字表の使ひ方は二つあります。

　一つ目は頁內檢索を掛けることです。これはウェブサイトの文章を讀んでいるときなど、調べたい字が文字データとしてコピーペースト出來る場合の方法です。Google 檢索をかけるだけでも上位に讀み方の情報はヒットしますが、檢字表を使ふことでより素早く多くの字を檢索し、彝文を讀み進めることが出來ます。

　二つ目は漢字字典の樣に、部首と畫數で字を引く方法です。紙の資料を讀んでゐる場合、動畫や畫像に表示された彝文を讀む場合は、こちらを使ふことになります。手順は次の通りです。

  1. まづ次高調（中昇調）の字の場合は上についてゐる倒置ブレーヴェを取り除きます。一覽性を高めるため檢字表には倒置ブレーヴェなしの字のみを載せてゐるためです。

  2. 字から部首を判斷し（判斷方法は後段「部首について」を參照）、頁先頭の部首目錄から該當する部首のセクションに飛びます。

  3. 字から部首を除いた畫數を數へ、該當する畫數のところを見ます。畫數の數へ方に迷ふ場合もありますが、本サイトでは一畫で書けさうなものは一畫として數へてゐます。目的の字が見當たらないときは前後の畫數も確認してみませう。

  4. 同じ畫數の字が並ぶ中から目的の字を探します。多くの場合は部首と畫數だけで充分に絞れますが、さうでない場合もあります。一番多いのは ꒓ (nyip) 部の2畫で32字あります。

  5. 次高調の字を引いてゐる場合は聲調を次高調 -x に變へて目的の音價を得ます。

　各字の後には拼音と國際音聲記號を載せてゐます。拼音と國際音聲記號の讀み方は適當にググれば出て來るので各自で調べてください。꒐ (qot) 部だけ國際音聲記號がありませんが、これは元データがさうなっていたためです。そのうち氣が向いたら自力で追補するかもしれません。頁末尾の次高調字一覽も國際音聲記號を未記載です。こちらも氣が向いたら以下略。

　なほ、讀みがわかったその單語の意味は、といふのはサポート外です。辭書にあたってください。

2. 配列について

　檢字表には字を部首と畫數順に並べてあります。同じ部首で同じ畫數の字は拼音順（＝ Unicode 順）ですが、抑も讀みがわからないから引く訣なのであまり意味はありません。一應拼音順を示しておくと、母音は i, ie, a, uo, o, e, u, ur, y, yr, 子音は（先頭に子音なし）, b, p, bb, nb, hm, m, f, v, d, t, dd, nd, hn, n, hl, l, g, k, gg, mg, hx, ng, h, w, z, c, zz, nz, s, ss, zh, ch, rr, nr, sh, r, j, q, jj, nj, x, y の順です。子音は漢語拼音の順序に手を加へたものになってゐます。母音は偶然か意圖的かわかりませんが、基本母音の順序→その他の母音→成節子音となってゐます。

　部首別の檢字表本文には次高調（中昇調）の字を載せてゐません。次高調は中平調の字の上に倒置ブレーヴェを載せて表記します。彝語で次高調が現れるのは中平調の語の異形態であることが多いためです。但し中平調の音節を使ふ語が無い vex, nzox, hnox の3字については低降調の ꃳ vep, ꋻ nzop, ꅸ hnop に倒置ブレーヴェを載せます。檢字の上では次高調字の音價は中平調字の音價から自明に知られるため、別立てで載せる必要がありません。同部首同畫數の字がなるべく少ない方が目で檢索しやすいので、本文からは次高調字を省いてゐます。

　但し Unicode 上では次高調字は別のコードポイントなのでそのままでは頁內檢索でヒットしません。このため頁の末尾に次高調字だけを別に列擧してゐます。これは頁內檢索に使ふものなので順番はどうでもいいのですが、部首と畫數は分けずに全て拼音順に並べてあります。

3. 部首について

　彝文字の部首は55あり、中には似てゐる部首もあります。部首の配列順（꒐、꒑、꒒…。部首目錄では縱に並べてゐます）は似てゐる部首が近くに來る樣になってゐるので慣れないうちは前後を見比べて判斷違ひをしてゐないか確かめませう。なほ部首は殆どが一畫なので漢字と違ひ部首の畫數順といふのはありません。

　部首を判斷する原則は單純です。

  • 字全體を上下に貫く要素のうち、最も左にあるもの

  • 字全體を左右に貫く要素のうち、最も上にあるもの

  • どちらもある場合は上下の要素を優先

  • どちらも無ければ最も左上にあるもの

꒨ 部は圓の外に畫が一切無い字から成ります。圓の外に畫があり、字全體を上下に貫いてゐる圓は ꒩ 部、さうでない圓は ꒪ 部です。但し ꆚ，ꉩ，ꋠ，ꍤ，ꑦ は ꒩ 部です。

　字によって微妙に形が變はる部首があります。

  • ꒑ の下の角が丸くなる。ꐱ，ꐳ など。

  • ꒕ は ꃷ で斜めに傾く。

  • ꒟，꒠ は左右反轉することがある。ꎍ，ꏸ など。

  • ꒤，꒳ は上下反轉することがある。ꋬ，ꁨ など。

  • ꒰ は自身の鏡像と一畫に融合することがある。ꈔ，ꐪ。

  • ꂚ，ꁱ は ꒙ 部。

  • ꌏ は ꒻ 部。

  • ꋧ は ꓆ 部。

　涼山規範彝文が制定された時に部首も整理制定され、表音文字で意味も關係ないので、漢字の部首の樣に恣意的に決まった例外はあまりありません。とはいへ原則にあてはまらないものもあり、以下に列擧します。

  • ꁌ pu は ꒐ 部。꒲ 部ではない。

  • ꀊ a は ꒐ 部。꒔ 部ではない。

  • ꀠ ba は ꒐ 部。꒳ 部ではない。

  • ꅎ ddup, ꌃ nzyt は ꒐ 部。꒰ 部ではない。

  • ꈞ gguot は ꒑ 部。꒢ 部ではない。

  • ꀆ ie, ꊵ ciep, ꋣ zzu は ꒓ 部。꒪ 部ではない。

  • ꐺ njuo は ꒓ 部。꒳ 部ではない。

  • ꅄ ddot は ꒓ 部。꒡ 部ではない。

  • ꆴ hlyp は ꒔ 部。꒢ 部ではない。

  • ꑇ njyp は ꒔ 部。꓆ 部ではない。

  • ꋻ nzop, ꋝ zzo は ꒕ 部。꒪ 部ではない。

  • ꇡ giep, ꅇ ddop は ꒕ 部。꒡ 部ではない。

  • ꈊ，ꌞ，ꏪ，ꏯ，ꊏ，ꊗ，ꋹ，ꎴ は ꒡ 部。꒳ 部ではない。

  • ꏠ jit は ꒢ 部の3畫。Noto Sans Yi を含む一部のフォントでは一畫目の ꒢ 部分が缺けてゐるが、Microsoft Yi Baiti を含む別のフォントでは ꏠ（←は環境により正しく表示されない場合がある）。

  • ꉤ nguot は ꒼ 部。꒢ 部ではない。

  • ꇾ kiep, ꂐ nbyr は ꒿ 部。꒐ 部ではない。

  • ꐒ qyt は ꓀ 部。꒐ 部ではない。

  • ꋖ zzie は ꓄ 部。꒢ 部ではない。

　ちなみに部首には元になった字（その部首に屬する字のうち、なるべく單純なもの）と同じ名前が付いてゐます。部首を入力する必要があれば（そんな機會があるのかわかりませんが）、彝語 IME では「部首名 + b」で入力できます。例へば ꒴ (nzup) なら nzupb と打鍵します。
