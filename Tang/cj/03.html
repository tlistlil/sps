基本字形（三）

  1. 1 X のキー
  2. 2 N のキー
  3. 3 Q のキー

前回、前々回の字形を見たらキーの所に指が動く様になりましたか？ 不安な人はくどい様ですが戻ってタイピング練習をもう一周しましょう。

今回は X, N, Q をやります。

X のキー

X は𘡃𘫺を入力します。A𘢌𘢏と紛らわしいので注意してください。複数のキーで分担する必要があるくらい西夏文字にはノとメのパターンが頻出するということです。画数が少なくアルファベットのXに近い方が𘡃𘫺と覚えましょう。

（本家倉頡では「難」ですが關係は無いようです。）

XAL 𗐌 wa¹ 姓の一つ「窪」
TXAL 𘋨 dzjwɨ¹ 「皇帝」
XAXX 𘔴 dju² 「産む」
TTX 𗐻 ˑjɨr² 「服」
TJLLX 𗚉 ·ju² 「民」
SXMBA 𘅋 ŋwer² 「膝、ひざまづく」
XBA 𗏎 bja² 「腹」
TJXBX 𗛎 dzjo² 「厩」
XTJL 𗏏 śjɨ̣¹ 「食べる」

N のキー

N は「フ」の字形を入力します。（本家倉頡の「弓」キーと同じ。）

ANB 𗬀 lwu² 「隠す」
ANLL 𗫱 khjã¹ 「危ない」
ANLL 𗢯 lhjwa¹ 「舌」
LNTM 𗀼 kạ² 「薬品秤り」
NXLNX 𗳤 lhjwɨ² 「中間」
ANX 𗣆 tshja² 漢語「千」などの音訳
BANNN 𘉌 zər² 「露（つゆ）」

Q のキー

Q は𘠩の字形を入力します。（本家倉頡の「手」キーと同じ。）

NNLQ 𘇜 be² 姓の一つ「皮」など
LSQ 𗀻 tshjwĩ¹ 「寸」漢語からの借用語
LNQA 𗁠 wow² 姓の一つ「王」など
ANLQ 𗣳 sjịj² 「書く、描く」
ALNQ 𗼋 dzjwɨ̣¹ 「敬う」

また、T𘠬が十字のJと組合せで𘡩になったように、Q𘠩と J の組合せで𘡡を入力します。

AQJ 𗢳 tha¹ 「仏」
QJA 𗘲 rjɨr² 「仏閣」この字は「仏」の左右を入替えています。この様な字を互換字といいます。
AQJA 𗦈 thew¹ 「浸透する」
XQJX 𗐆 tśjọ¹ 「醜い」
